# 简介

## 相关功能

写一个类似微信的渲染消息流的组件，支持不同类型：
1.文本
2.图片 
3.系统消息，例如加某个人到群组里，撤销消息 
要求考虑可扩展性，具体UI可以简单实现

**重点考察组件设计和扩展性，比如：新增一个消息类型不会改动现有的代码等**
